# 🎯 Sentiment Analysis on Text Data

## 📌 Overview

This project focuses on **Sentiment Analysis of text data** using Machine Learning techniques. The goal is to classify input text (such as movie reviews) into sentiment categories like **positive** or **negative**.

The dataset utilized for model training is IMDB movie review dataset freely avaliable on kaggle.

The system includes:

* Data preprocessing
* Feature extraction
* Model training
* Prediction interface

---

## 🚀 Functionality

* Text preprocessing (cleaning, tokenization, normalization)
* Feature extraction using NLP techniques
* Machine Learning model for sentiment classification
* Real-time prediction using a simple interface
* Lightweight and easy to deploy

---

## 🧠 Working

1. Input text is taken from the user
2. Text is preprocessed:

   * Lowercasing
   * Removing stopwords
   * Tokenization
3. Features are extracted (e.g., Bag of Words / TF-IDF)
4. Trained model predicts sentiment
5. Output is displayed to the user
