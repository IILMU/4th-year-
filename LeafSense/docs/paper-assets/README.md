Figures and plots for the IEEE-style LeafSense paper (LaTeX `\includegraphics` paths).

Point your `.tex` file here, for example:

- `docs/paper-assets/fig2.jpeg` — model architecture
- `docs/paper-assets/fig3.jpeg` — inference pipeline
- `docs/paper-assets/dataset_split.png` — dataset split visualization
- `docs/paper-assets/fig5.png`–`fig7.png` — metrics / confusion / ROC

These assets are not loaded by the Flask or React runtime.
