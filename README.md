# 🎓 IILM University | 4th Year Repository (2022-2026)

Welcome to the central repository for the 4th-year cohort at IILM Greater Noida. This is a collaborative space designed to archive our collective academic work, technical projects, and social media presence.

---

## 📂 Directory Overview

* **`📂 /Projects`**: Major and minor project source code and documentation.
* **`📂 /Social-Media`**: 
    * `📸 /Graphics`: Posters, event flyers, and reel assets.
    * `✍️ /Content`: Captions, hashtags, and LinkedIn templates.
* **`📂 /Academic-Resources`**: Notes, question banks, and lab manuals.
* **`📂 /Placement-Portal`**: Resume templates, coding practice, and interview logs.
* **`📂 /Events`**: Coverage of hackathons, fests, and guest lectures.

---

## 🤝 Community Guidelines

As a contributor, please ensure this repository remains organized for everyone:

1.  **Uniform Naming**: Use clear titles like `SocialMedia_Workshop_Oct2025`.
2.  **No Redundancy**: Check if a folder already exists before creating a duplicate.
3.  **Media Quality**: Upload high-resolution assets to the `/Social-Media` folder.
4.  **Privacy**: Do not upload personal sensitive data; keep it professional.
